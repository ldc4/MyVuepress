# 常用快捷键


```
// mac键盘
⇧：shift  
^：ctrl  
⌥：option  
⌘：command  
```

## 搜索框

> `⌘` `P`

在`⌘` `P`模式，输入`>`，可以搜索命令（等价于 `⇧` `⌘` `P` 或 `F1`）

在`⌘` `P`模式，输入`#`, 可以跳转到当前工作区下Symbol - 变量或函数（等价于`⌘` `T`）

在`⌘` `P`模式，输入`:`，可以跳转到当前编辑器下指定行（等价于 `^` `G`）

在`⌘` `P`模式，输入`@`，可以跳转到当前编辑器下Symbol - 变量或函数（等价于 `⇧` `⌘` `O`）, 进一步输入`:`，则可以进行分类
